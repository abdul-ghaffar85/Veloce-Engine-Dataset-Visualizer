# Installation Guide

Follow these steps to set up the Veloce Engine application locally.

## Prerequisites
- Python 3.12+
- Node.js (v18+)
- npm or yarn

## Backend Setup

1. **Activate the virtual environment**:
   ```bash
   .venv\Scripts\activate
   ```
2. **Install dependencies**:
   ```bash
   python -m pip install -r requirements.txt
   ```

## Frontend Setup

1. **Navigate to the frontend directory**:
   ```bash
   cd frontend
   ```
2. **Install dependencies**:
   ```bash
   npm install
   ```
