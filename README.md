# Veloce Engine

Veloce Engine is an AI-powered data analytics platform for CSV and Excel files. It automatically transforms uploaded datasets into profiling insights, relationships, aggregations, chart recommendations, and interactive dashboards, all running fully in-memory with a FastAPI backend and a sleek React frontend.

## Features
- **File Upload**: Seamless drag-and-drop CSV/Excel uploads.
- **Profiling Engine**: Automatic semantic type inference and outlier detection.
- **Relationship & Aggregation Engine**: Intelligent analysis of DataFrames.
- **Dynamic Dashboards**: Auto-generated visualizations using Chart.js and AG Grid.
- **Premium Aesthetics**: High-end dark mode interface.
